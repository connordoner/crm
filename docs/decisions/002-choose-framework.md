# ADR 2: Choose application framework

Date: 26 November 2023

## Status

Accepted

## Context

I need to build my personal CRM with a framework to allow me to focus on building it rather than implementing functionality common to all web applications.

## Decision

I'll use Ruby on Rails.

## Consequences

* I now have to program primarily in Ruby
* Options for my database engine are now limited to PostgreSQL, SQLite, MySQL, SQL Server or Oracle Database
